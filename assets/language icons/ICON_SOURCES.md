# Skill icon sources

All generated PNG files are 128 × 128 with transparent backgrounds.
Product names and logos remain trademarks of their respective owners.

- [Devicon](https://devicon.dev/) (MIT): Android, Astro, AWS, Cloudflare,
  cPanel, DigitalOcean, Docker, Fastify, Git, Kotlin, Laravel, Linux, MongoDB,
  Next.js, Nginx, Python, React, Redis, Slack, Swift, Twilio, WooCommerce, and
  WordPress.
- [Simple Icons](https://simpleicons.org/) (CC0 1.0): ElevenLabs, QuickBooks,
  Squarespace, and Stripe.
- [Font Awesome Free](https://fontawesome.com/) (CC BY 4.0): generic symbols
  used for AI, Assembly, DNS, REST, Shell, and VPS. Copyright Fonticons, Inc.
- [LobeHub Icons](https://lobehub.com/icons/openai) (MIT): OpenAI.
- [WebRTC press kit](https://webrtc.github.io/webrtc-org/press/): WebRTC.
- [Devicons](https://devicons.io/icons/websocket/) (MIT): WebSocket.
- User-provided artwork: Cloudflare R2 and XBRL.

## Notes

- AI, Assembly, DNS, REST, Shell, and VPS use generic technical symbols because
  these concepts do not have official product logos.
- WebSocket uses a community icon because the protocol has no official brand
  logo.
